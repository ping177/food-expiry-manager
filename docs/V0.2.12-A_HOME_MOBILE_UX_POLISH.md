# v0.2.12-A 首页 Mobile UX Polish

## 目标

将登录信息从库存首页顶部迁移到“我的”页面，并提供固定底部导航：库存、居中的新增商品入口、我的；导航使用内置 SVG 图标。

## 范围

- 库存页保留商品列表、搜索和筛选；新增商品仅从底部导航的无文字 `+` 入口进入。
- 我的页展示当前账号状态、脱敏邮箱、退出登录和三个未来设置占位项。
- 添加商品、库存详情和编辑属于任务流，不显示底部 Tab 导航。
- 底部导航使用页面背景、细分隔线、SVG 图标和短文字标签；默认状态统一灰色，选中 Tab 使用现有绿色主题和 underline，不使用独立白色圆角卡片或阴影。
- 底部导航和内容区域均使用移动端安全区留白。
- 库存首页只保留“库存”标题，不展示产品名或批次排序说明；搜索、筛选和列表不变。

## 非目标

- 不新增第三个 Tab。
- 不修改 Supabase Auth flow、schema、RLS、环境变量、库存数据模型或库存 API。
- 不实现提醒设置、数据导出或偏好设置页面。
